# Vibe Coding 学习教程

这是一个用于学习和记录 Vibe Coding、Cursor 等 AI 编程工具使用方法的教程仓库。

## 📚 目录结构

```
vibe_coding/
├── tutorials/          # 教程内容
│   ├── vibe-coding/   # Vibe Coding 相关教程
│   └── cursor/        # Cursor 相关教程
└── examples/          # 示例代码
```

## 🎯 内容说明

### Tutorials（教程）
- **vibe-coding/**: 存放 Vibe Coding 相关的学习教程和笔记
- **cursor/**: 存放 Cursor IDE 相关的学习教程和笔记

### Examples（示例）
- 存放各种示例代码和实践项目

## 🚀 使用说明

1. 浏览 `tutorials/` 目录下的相关教程
2. 查看 `examples/` 目录下的示例代码
3. 根据需要添加自己的学习笔记和代码
